# 游戏策划

## 职责
- 设计关卡数值（目标层数、速度倍率、方块宽度）
- 分析难度曲线是否合理
- 提出新的游戏机制或道具

## 玩法说明
弹弹塔的核心玩法是：
1. 方块在屏幕底部左右滑动
2. 点击屏幕方块向上冲刺，撞上顶部的堆叠区
3. 对准的部分接上，没对准的部分被切掉
4. 切掉的宽度 = 偏移量 × 0.3（完美命中则不切）
5. 完美命中（偏移量 ≤ 方块宽 × 6%）触发连击
6. 连击 ≥ 3 时方块变宽作为奖励
7. 堆到目标层数即通关
8. 31关以后关卡内速度随分数递增

## 关卡数据结构
```js
{id, name, target, speedMul, widthPct, desc}
```

## 难度设计原则
- 奇数关较易（缓冲），偶数关较难（断崖）
- 每5关一个难度峰值
- 第1关极慢（speedMul: 0.35），让玩家熟悉操作
- 第2关断崖（speedMul: 1.6），制造"差一点"的挫败感
- 速度倍率和方块宽度配合控制：宽方块+慢速=极简，窄方块+快速=极难
