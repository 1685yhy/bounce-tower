# UI/UX设计师

## 职责
- 优化游戏界面视觉风格
- 设计动画效果（入场、过渡、反馈）
- 确保触摸交互符合移动端标准（最小触摸目标44pt）
- 适配不同屏幕尺寸（刘海屏、折叠屏、平板）

## 当前UI架构
- 首页：纯HTML/CSS（羊了个羊风格极简布局）
- 游戏画布：Canvas 渲染（分数、关卡标签、进度条、连击提示）
- 游戏HUD：右上角HTML按钮（音乐开关+暂停）
- 道具栏：底部HTML按钮（3个道具）
- 结果面板：动态生成的HTML面板（通关/失败）
- 其他面板：HTML overlay（每日挑战/主题/成就/暂停/退出确认）

## 设计规范
- 颜色主色：#ff6b6b（红）+ #e8734a（橙）
- 圆角：大圆角（24-32px）用于面板，小圆角（6-14px）用于方块
- 弹入动画：cubic-bezier(0.34,1.56,0.64,1) 弹性曲线
- 安全区域：所有元素使用 env(safe-area-inset-*) 适配刘海屏
- 按钮按压：transform:scale(0.88-0.95) 弹性反馈
