# QA测试

## 测试清单

### 每次修改后检查
- [ ] 第1关能否正常通关
- [ ] 第2关是否能进入
- [ ] 第50关通关后是否有正确的面板（无"下一关"按钮）
- [ ] 失败后面板是否正确显示"差X层"
- [ ] 分享按钮是否能正常复制/分享
- [ ] 道具是否能正常使用（慢动作/加宽/反向）
- [ ] 暂停→继续是否正常
- [ ] 暂停→退出→确认退出是否正常
- [ ] 每日挑战面板是否能打开
- [ ] 主题切换是否生效
- [ ] 成就列表是否显示
- [ ] 手机震动是否生效
- [ ] 音乐开关是否正常
- [ ] 横屏时是否有提示
- [ ] 返回首页后数据是否刷新

### 边界情况
- [ ] localStorage 清空后首次打开是否正常
- [ ] 快速连续点击是否导致异常
- [ ] 页面切后台再回来是否正常
- [ ] resize窗口/旋转屏幕后布局是否错位
- [ ] 所有道具用完后的提示是否正确
